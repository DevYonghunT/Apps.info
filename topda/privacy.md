# 톺다(Topda) 개인정보처리방침

**시행일:** 2026-03-26  
**최종 수정일:** 2026-04-03

---

## 1. 수집하는 개인정보 항목

톺다(이하 "서비스")는 다음과 같은 개인정보를 수집합니다.

| 구분 | 수집 항목 | 수집 방법 |
|------|----------|----------|
| 필수 | 이메일 주소 | 회원가입 시 입력 |
| 선택 | 학교명, 학년 | 프로필 설정 시 입력 |
| 자동 | 학습 질문 텍스트, 학습 이력, 블룸 분류 결과 | 서비스 이용 시 자동 생성 |

## 2. 개인정보 수집 및 이용 목적

- 학습 진도 관리 및 학습 분석 리포트 생성
- AI 학습 도우미 서비스 제공
- 교사의 학생 학습 현황 파악
- 서비스 품질 개선

## 3. 개인정보 보유 및 이용 기간

- 회원 탈퇴 시까지 보유하며, 탈퇴 즉시 파기합니다.
- 관련 법령에 의한 보존 필요 시 해당 기간 동안 보관합니다.
  - 서비스 이용 기록: 3개월 (통신비밀보호법)

## 4. 개인정보의 해외 이전

서비스는 AI 응답 생성을 위해 학습 질문 텍스트를 해외로 전송합니다.

| 항목 | 내용 |
|------|------|
| 이전 국가 | 미국 |
| 수취자 | Google LLC |
| 이전 항목 | 학습 질문 텍스트 (익명화 처리 후 전송) |
| 이전 목적 | Google Gemini AI를 통한 학습 설명 생성 |
| 보유 기간 | API 처리 완료 즉시 삭제 |

> **참고:** 학생의 이름, 학번 등 개인식별정보는 전송 전 자동 제거(익명화)됩니다.

## 5. 개인정보보호 책임자

- **이메일:** privacy@topda.kr

## 6. 정보주체의 권리

이용자는 다음과 같은 권리를 행사할 수 있습니다.

1. **열람권:** 수집된 개인정보의 열람을 요청할 수 있습니다.
2. **정정권:** 부정확한 개인정보의 정정을 요청할 수 있습니다.
3. **삭제권:** 개인정보의 삭제를 요청할 수 있습니다.
4. **처리정지권:** 개인정보 처리의 정지를 요청할 수 있습니다.

권리 행사는 privacy@topda.kr로 요청하시면 10일 이내에 처리합니다.

## 7. 개인정보의 안전성 확보 조치

- 비밀번호 암호화 저장 (Supabase Auth)
- HTTPS 통신 암호화
- 접근 권한 관리 (Row Level Security)
- 개인정보 처리 시스템 접근 로그 관리

## 8. AI 데이터 처리 안내

본 앱은 학습 질문 텍스트를 Google Gemini API에 전송하여 AI 응답을 생성합니다.

- AI 응답은 자동 생성된 것으로, 정확성을 보장하지 않습니다.
- 모든 AI 응답에는 "AI 생성" 라벨이 표시됩니다.
- 교사가 AI 응답을 검토하고 부적절한 답변을 플래그할 수 있습니다.
- AI에 전송되는 데이터에는 개인식별정보가 포함되지 않습니다.
