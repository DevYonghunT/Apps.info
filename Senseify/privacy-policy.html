개인정보처리방침 / Privacy Policy

Senseify는 사용자의 개인정보를 수집하지 않습니다.

모든 센서 데이터는 기기 내에서만 처리되며, 외부 서버로 전송되지 않습니다.

1. 수집하는 개인정보

Senseify는 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

2. 센서 데이터 처리

앱에서 사용하는 센서 데이터(가속도, 자이로스코프, 자기장, 기압, 소리, GPS, 조도)는:

  • 기기 내에서만 실시간 처리됩니다
  • 외부 서버로 전송되지 않습니다
  • 앱 종료 시 자동으로 삭제됩니다
  • CSV 내보내기는 사용자의 명시적 조작에 의해서만 수행됩니다

3. 위치 정보

GPS 센서 기능 사용 시 위치 정보에 접근하지만, 이 데이터는 기기 내에서만 처리되며 어디에도 저장되거나 전송되지 않습니다.

4. 마이크 접근

소리 측정 기능 사용 시 마이크에 접근하지만, 음성은 녹음되지 않으며 소리 크기(dB)와 주파수 분석에만 사용됩니다.

5. 제3자 서비스

Senseify는 광고, 분석 도구, 제3자 SDK를 사용하지 않습니다.

6. 아동 개인정보 보호

Senseify는 교육용 앱으로, 개인정보를 수집하지 않으므로 모든 연령의 사용자가 안전하게 사용할 수 있습니다.

7. 변경 사항

본 개인정보처리방침이 변경될 경우 이 페이지를 통해 공지합니다.

Senseify does not collect any personal information.

All sensor data is processed locally on your device and is never transmitted to external servers.

1. Information We Collect

Senseify does not collect, store, or transmit any personal information.

2. Sensor Data Processing

Sensor data used by the app (accelerometer, gyroscope, magnetometer, barometer, microphone, GPS, light) is:

  • Processed only on your device in real-time
  • Never transmitted to external servers
  • Automatically deleted when the app is closed
  • CSV export is performed only by explicit user action

3. Location Data

The GPS feature accesses location data, but this data is processed only on the device and is not stored or transmitted anywhere.

4. Microphone Access

The sound measurement feature accesses the microphone, but audio is not recorded. It is used only for sound level (dB) and frequency analysis.

5. Third-Party Services

Senseify does not use any advertisements, analytics tools, or third-party SDKs.

6. Children's Privacy

Senseify is an educational app that does not collect personal information, making it safe for users of all ages.

7. Changes

Any changes to this privacy policy will be posted on this page.
